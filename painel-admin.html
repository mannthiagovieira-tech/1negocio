Como enviar o laudo
Processo atual (manual):
1. Cliente paga R$99 via Stripe → aparece aqui com status Aguardando envio
2. Abra o diagnóstico do cliente no Supabase ou por e-mail
3. Gere o PDF do laudo usando a URL /laudo-preview.html?id=NEGOCIO_ID
4. Envie por e-mail para o cliente e marque como Concluído

Automação futura: Webhook Stripe → Edge Function → geração automática do PDF → envio via Resend em <15min.
